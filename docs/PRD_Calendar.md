# PRD: 캘린더 Feature

> 로아 캘린더의 핵심 기능. 구글 캘린더 수준의 일정 관리를 제공한다.

---

## 1. 개요

사용자가 일정을 생성, 조회, 수정, 삭제할 수 있는 캘린더 기능.
월간/주간/일간/연간 4가지 뷰를 제공하며, 반복 일정, 다중 캘린더, 알림을 지원한다.

### 2.3 알림

| # | 필드 | 타입 | 필수 | 설명 |
|---|------|------|------|------|
| 8 | 알림 | [Reminder] | - | 최대 5개 |

#### Reminder 타입 (Core 레이어 공유)

| 필드 | 타입 | 설명 |
|------|------|------|
| id | UUID | 고유 ID |
| type | ReminderType | `.relative` (N분 전) / `.absolute` (특정 시각) |
| offsetMinutes | Int? | relative 시 시작 시간 기준 N분 전 |
| absoluteDate | Date? | absolute 시 특정 날짜/시간 |

Calendar과 Todo가 동일한 Reminder 타입을 공유한다.

#### 알림 시간 옵션

- 이벤트 시작 시: 0분 전
- 5분 전 / 10분 전 / 15분 전 / 30분 전
- 1시간 전 / 2시간 전
- 1일 전 / 2일 전 / 1주 전
- 커스텀 (N분/시간/일/주 전)

#### 알림 제한 사항

- iOS 로컬 알림 전체 64개 제한
- 스케줄링 전략: 향후 7일 이내의 알림만 등록
- 앱 포그라운드 복귀 시 알림 재스케줄링

## 3. 캘린더 뷰 (4개)

### 3.1 연간 뷰

- 12개월 그리드 표시
- 일정이 있는 날짜에 점(dot) 표시
- 월 탭하면 월간 뷰로 이동

### 3.2 월간 뷰

- 달력 그리드 (6주 고정 또는 가변)
- 각 날짜에 일정 최대 3개 미리보기 (색상 바)
- 3개 초과 시 "+N개 더보기" 표시
- 날짜 탭하면 일간 뷰로 이동
- 좌우 스와이프로 월 이동
- 한국 공휴일 빨간색 표시
- 오늘 날짜 하이라이트

### 3.3 주간 뷰

- 7일 x 24시간 타임 그리드
- 상단: 종일 이벤트 스트립
- 이벤트를 색상 블록으로 표시
- 겹치는 이벤트 나란히 배치 (side-by-side 레이아웃)
- 현재 시간 빨간 표시선
- 좌우 스와이프로 주 이동
- 좌측 미니 캘린더 (월간 미니맵)

### 3.4 일간 뷰

- 24시간 타임 그리드
- 상단: 종일 이벤트 스트립
- 이벤트를 색상 블록으로 상세 표시 (제목, 시간, 위치)
- 겹치는 이벤트 나란히 배치
- 현재 시간 빨간 표시선
- 좌우 스와이프로 일 이동

## 4. 캘린더 기능 (12개)

### 4.1 일정 CRUD

| 기능 | 설명 |
|------|------|
| 생성 | 빈 시간 탭 → 해당 시간 프리필 / "+" 버튼 → 빈 폼 |
| 조회 | 일정 탭 → 상세 화면 |
| 수정 | 상세 화면 → 편집 모드 |
| 삭제 | 스와이프 삭제 또는 상세 화면에서 삭제 |
| 복사 | 상세 화면 → 이벤트 복사 (Duplicate) |

### 4.2 드래그 인터랙션

| 기능 | 뷰 | 설명 |
|------|-----|------|
| 드래그로 이동 | 주간/일간 | 이벤트를 드래그하여 시간/날짜 변경 |
| 드래그로 크기 조절 | 주간/일간 | 하단 핸들 드래그하여 시간 조절 |
| 스냅 | 주간/일간 | 15분 단위 스냅 |

### 4.3 겹침 레이아웃

- 같은 시간대 이벤트가 2개 이상이면 가로 분할
- 2개: 50% / 50%
- 3개: 33% / 33% / 33%
- 4개 이상: 스크롤 또는 축소

### 4.4 현재 시간 표시선

- 일간/주간 뷰에 빨간 수평선
- 실시간 업데이트 (Timer)
- 앱 진입 시 현재 시간 위치로 자동 스크롤

### 4.5 검색

- 전체 텍스트 검색 (제목, 위치, 메모, URL)
- 검색 결과를 리스트로 표시
- 결과 탭하면 해당 일정으로 이동

### 4.6 날짜 이동 (Jump to Date)

- 날짜 선택 피커
- 선택 시 해당 날짜의 현재 뷰로 이동
- "오늘" 버튼으로 현재 날짜 복귀

### 4.7 한국 공휴일 캘린더

- 대한민국 공휴일 내장 (법정 공휴일)
- 음력 기반 공휴일 포함 (설날, 추석, 부처님오신날)
- 공휴일은 빨간색, 별도 캘린더로 분류
- 사용자가 표시/숨김 토글 가능

### 4.8 첫 요일 설정

- 기본값: 기기 로캘 따름 (한국 = 월요일)
- 설정에서 일요일/월요일 선택 가능
- 모든 뷰에 즉시 반영

### 4.9 미니 캘린더

- 주간/일간 뷰에서 상단 또는 사이드에 소형 월간 달력 표시
- 날짜 탭하면 해당 날짜로 이동
- 일정 있는 날짜에 점(dot) 표시

### 4.10 미완료 Task 롤오버

- 마감일이 지난 미완료 Task는 오늘 날짜에 표시
- 별도 스타일로 구분 (오버듀 표시)
- 완료하거나 날짜를 변경할 때까지 유지

### 4.11 .ics 내보내기/가져오기

- 내보내기: 단일 일정 또는 캘린더 전체를 .ics 파일로 공유
- 가져오기: .ics 파일 열기 → 일정 추가
- iOS 공유 시트 연동

### 4.12 다중 캘린더 관리

- 캘린더 생성/수정/삭제
- 캘린더별 색상 지정
- 캘린더별 표시/숨김 토글
- 기본 캘린더 설정

## 6. 화면 흐름

```
탭바 [캘린더]
  └─ 캘린더 메인 (기본: 월간 뷰)
      ├─ 뷰 전환 (연간/월간/주간/일간)
      ├─ 검색
      ├─ 날짜 이동
      ├─ 캘린더 필터 (표시/숨김)
      ├─ 일정 탭 → 일정 상세
      │   ├─ 편집
      │   ├─ 삭제
      │   └─ 복사
      └─ "+" 또는 빈 시간 탭 → 일정 생성
          └─ 저장 → 캘린더 메인으로 복귀
```

## 7. 위젯

| 위젯 | 크기 | 내용 |
|------|------|------|

## 9. 비기능 요구사항

- CLAUDE.md의 UX 품질 기준 준수 (햅틱, 애니메이션, 3상태 처리, 다크모드, 접근성)
- 성능 기준: Cold Launch 1초, 화면 전환 0.3초, 메모리 100MB
- 날짜 연산: `Calendar(identifier: .gregorian)` + `Locale(identifier: "ko_KR")` 필수
- 알림 스케줄링: 향후 7일 이내만 등록, 포그라운드 복귀 시 재스케줄링
