# PRD: 자기관리 Feature

> 로아 캘린더의 자기관리 기능. Calendar + Pomodoro + Todo 3개 데이터를 종합 분석하여 다른 앱에서 불가능한 인사이트를 제공한다.

---

## 1. 개요

캘린더, 뽀모도로, Todo 데이터를 크로스 분석하여 일일 성과 점수, 번아웃 경고, 시간 분석을 제공한다.
습관 트래커, 에너지 체크인, 일일/주간 리뷰, 분기 OKR을 통해 장기적 자기관리를 지원한다.

---

## 2. 화면 구조

### 2.1 대시보드 (기본 레이아웃)

탭바 [자기관리] 진입 → 스크롤 대시보드:

```
┌─ 일일 성과 점수 (큰 숫자 + 30일 트렌드)
├─ 오늘 요약 카드 (에너지, 집중, 완료율)
├─ 번아웃 경고 카드 (조건 충족 시만)
├─ 섹션 네비게이션
│   ├─ 통계
│   ├─ 리뷰
│   ├─ 목표
│   └─ 습관
└─
```

### 2.2 커스터마이징

- 사용자가 카드 순서 변경 가능 (드래그)
- 카드 표시/숨김 토글 가능
- 섹션 순서 변경 가능
- 설정에서 기본 레이아웃 복원 가능

## 3. 크로스 도메인 인텔리전스 (A)

> 데이터 접근은 Core 서비스를 통해서만 한다:
> - CalendarQueryService — 이벤트 조회, 참석 여부 집계
> - PomodoroLogService — 세션 기록 조회, 시간대별 집계
> - TaskService — Task 완료율 조회
> - EnergyService — 에너지 체크인 기록/조회, 습관 상관분석

### 3.1 일일 성과 점수

#### 공식

| 도메인 | 비중 | 계산 |
|--------|------|------|
| 뽀모도로 | 40% | 완료 세션 / 예정 세션 |
| Todo | 40% | 오늘 완료 Task / 오늘 마감+예정 Task |
| 캘린더 | 20% | 참석한 이벤트 / 예정 이벤트 |

#### 규칙

- 해당 도메인에 활동 없으면 비중 제외 후 재계산
- 모든 도메인 활동 없는 날은 점수 null (0%가 아님)
- 30일 트렌드 차트 표시
- 주간 평균 / 월간 평균 표시

#### 표시

- 대시보드 최상단에 큰 숫자 (예: 78%)
- 색상: 80%+ 초록, 60~80% 노랑, 60% 미만 빨강
- null일 때 "활동 없음" 표시 (0% 아님)

### 3.2 시간대별 집중 품질 히트맵

- 24시간 x 7요일 히트맵
- 뽀모도로 세션 완료 데이터 기반
- 색상 농도 = 해당 시간대 완료 세션 수
- 피크 시간대 하이라이트 + 추천 메시지 ("오전 10~12시가 가장 집중력 높은 시간입니다")
- 최근 30일 데이터 기준

### 3.3 아이젠하워 시간 분배

- 원형 차트 또는 4분할 막대 차트
- 각 분면별 실제 투입 뽀모도로 시간 (분)
- Q1(즉시 실행) / Q2(일정 잡기) / Q3(위임) / Q4(제거)
- 주간/월간 탭 전환
- Q2 비율 목표: 60% 이상 권장 (비율 미달 시 안내 메시지)

### 3.4 계획 vs 실행 갭

- 주간 차트: 캘린더 예정 블록 수 vs 실제 완료 Task/세션 수
- 과잉 계획 패턴 감지: "이번 주 계획 대비 실행률 45% — 일정을 줄여보세요"
- 과소 계획 패턴 감지: "이번 주 계획 없이 실행한 Task가 70% — 계획을 세워보세요"
- 요일별 실행률 차트

### 3.5 번아웃 조기 경고

#### 알고리즘

7일 롤링 평균 기준, 아래 조건 3개 중 **2개 이상 동시 충족** 시 경고:

| 조건 | 기준 |
|------|------|
| 일일 성과 점수 | 7일 평균 60% 미만 |
| 에너지 체크인 | 7일 평균 2.5/5 미만 |
| 뽀모도로 세션 | 전주 대비 30% 이상 감소 |

#### 경고 방식

- 비알람형 인앱 카드: "최근 2주 패턴을 확인해보세요"
- 하루 최대 1회 표시
- 탭하면 상세 분석 화면으로 이동
- 푸시 알림 아님 (앱 내에서만)

## 4. 셀프 리뷰 시스템 (B)

### 4.1 하루 마감 리뷰

#### 트리거

- 사용자 설정 퇴근 시간 30분 전 로컬 알림 (기본: 오후 6시)
- 시간 커스텀 가능
- 알림 권한 거부 시 앱 진입 시 인앱 배너

#### 화면 구성

1. **자동 요약** (데이터에서 자동 생성)
   - 오늘 완료 Task 수 / 전체
   - 뽀모도로 세션 수 + 총 집중 시간
   - 캘린더 이벤트 수

2. **3문항** (선택 입력)
   - "오늘 잘한 점은?"
   - "무엇이 막혔나?"
   - "내일 최우선 과제는?"

3. **오늘의 성과 (Wins Log)**
   - "오늘의 성과" 1개 입력 (소프트 강제)
   - 미입력 시 리뷰 스트릭 끊김 경고만 표시
   - 앱 사용 차단 없음

#### 저장

- 날짜별 리뷰 기록으로 저장
- 검색 가능
- Logbook에서 과거 리뷰 조회

### 4.2 주간 리뷰

#### 트리거

- 매주 일요일 오전 알림 (기본, 커스텀 가능)
- 수동 실행도 가능

#### 화면 구성

1. **자동 집계**
   - 이번 주 Task 완료율
   - 이번 주 총 집중 시간 + 세션 수
   - 이번 주 에너지 평균
   - 이번 주 일일 성과 점수 평균
   - 목표(OKR) 진행률

2. **미완료 Task 처리**
   - 미완료 Task 리스트 표시
   - 각 Task에 대해: 다음 주 이월 / Someday / 삭제 선택

3. **주간 한 줄 자동 요약**
   - "이번 주: 12세션, 34태스크 완료(82%), 에너지 평균 3.4/5"
   - 사용자가 한 줄 코멘트 추가 가능
   - 저장하여 연간 타임라인 생성

### 4.3 에너지 체크인

#### 데이터 모델

| 필드 | 타입 | 설명 |
|------|------|------|
| id | UUID | 고유 ID |
| timestamp | Date | 기록 시각 |
| level | Int | 1~5 (1=매우 낮음, 5=매우 높음) |
| note | String? | 선택 메모 |
| trigger | EnergyTrigger | `.scheduled` / `.postPomodoro` / `.manual` |

#### 동작

- 기본: 하루 3회 예약 알림 (오전 9시, 오후 1시, 오후 6시)
- 시간 커스텀 가능
- 뽀모도로 세션 완료 후 선택적 체크인 (설정에서 ON/OFF)
- 수동으로 언제든 입력 가능
- 2초 소요: 이모지 5개 중 탭
- 알림 권한 거부 시 앱 진입 시 인앱 배너

#### 분석

- 일간/주간/월간 에너지 차트
- 요일별 에너지 패턴
- 시간대별 에너지 패턴
- 습관-에너지 상관분석 (§5.5)과 연동

### 4.4 승리 기록 (Wins Log)

- 하루 마감 리뷰(§4.1)에서 입력
- 별도 화면에서 월간 승리 리스트 조회
- 월초에 "지난달 성과" 자동 요약 카드 표시
- 검색 가능

## 5. 습관 트래커 (D)

### 5.1 Habit 데이터 모델 (별도 @Model)

| 필드 | 타입 | 설명 |
|------|------|------|
| id | UUID | 고유 ID |
| title | String | 습관 이름 (필수) |
| emoji | String? | 아이콘 이모지 |
| color | EventColor | 색상 |
| frequency | HabitFrequency | `.daily` / `.weekly(daysOfWeek: [Int])` |
| reminderTime | Date? | 알림 시간 |
| createdAt | Date | 생성일 |
| isArchived | Bool | 보관 여부 |

### 5.2 HabitCompletion 모델

| 필드 | 타입 | 설명 |
|------|------|------|
| id | UUID | 고유 ID |
| habit | Habit | 소속 습관 |
| date | Date | 완료 날짜 |
| completedAt | Date | 완료 시각 |

### 5.3 습관 기능

#### 생성/관리

- 커스텀 습관 생성 (제목, 이모지, 색상, 빈도, 알림)
- 습관 편집/보관/삭제
- 보관된 습관은 목록에서 숨김 (데이터 유지)
- 알림 설정 습관 최대 5개 (알림 예산 제한). 초과 시 "알림 슬롯이 부족합니다" 안내

#### 완료 체크

- 원탭 완료 (습관 목록에서)
- 캘린더 뷰에서도 습관 표시 (점 또는 아이콘)
- 이미 완료한 습관 탭하면 취소 가능

#### 스트릭

- 연속 달성 일수 표시
- 현재 스트릭 / 최장 스트릭
- daily: 매일 완료해야 유지
- weekly: 지정된 요일에 완료해야 유지

#### 연간 히트맵

- 최근 52주 표시 (GitHub 잔디 스타일)
- 색상 농도 = 해당 날짜 완료 여부
- 습관별 개별 히트맵

### 5.5 습관-에너지 상관분석

- 충분한 데이터(최소 14일) 이후 자동 분석
- "운동한 날 에너지 평균 4.2, 안 한 날 2.8" 형태
- 각 습관별 에너지 상관 수치 표시
- 양의 상관 / 음의 상관 / 무관 분류

## 6. 목표 관리 (C)

### 6.1 분기 OKR

#### Objective 모델

| 필드 | 타입 | 설명 |
|------|------|------|
| id | UUID | 고유 ID |
| title | String | 목표 이름 |
| quarter | String | 분기 (예: "2026-Q2") |
| status | OKRStatus | `.active` / `.completed` / `.archived` |
| createdAt | Date | 생성일 |

#### KeyResult 모델

| 필드 | 타입 | 설명 |
|------|------|------|
| id | UUID | 고유 ID |
| title | String | 핵심 결과 |
| objective | Objective | 소속 목표 |
| targetValue | Double? | 목표값 (선택, 수치형 KR) |
| currentValue | Double? | 현재값 (수치형 KR) |
| progressByTodos | Double | Todo 기반 자동 진행률 (완료/전체) |
| grade | Double? | 분기 말 평가 (0.0~1.0) |

#### OKR-to-Todo 관계

- 하나의 Todo는 하나의 Key Result에만 연결 (N:1)
- 하나의 Key Result에 여러 Todo 연결 가능
- Todo 삭제 시 Key Result 진행률 자동 재계산
- progressByTodos = 연결된 Todo 완료 수 / 전체 수

#### 제한

- 분기당 최대 3개 Objective
- Objective당 최대 4개 Key Result
- 하위 Objective, 이니셔티브 등 추가 계층 없음 (v1)

#### 분기 리뷰

- 분기 말 각 Key Result에 0.0~1.0 평가
- 자동 집계: 투입 뽀모도로 시간, 연결 Task 완료율
- 다음 분기 OKR 생성 유도

### 6.2 생활 영역 균형 (Wheel of Life)

#### 기본 영역 (6개 고정, 라벨 사용자 변경 가능)

| 영역 | 기본 라벨 |
|------|----------|
| 1 | 건강 |
| 2 | 업무/커리어 |
| 3 | 관계 |
| 4 | 학습/성장 |
| 5 | 재정 |
| 6 | 여가/취미 |

#### 동작

- 월 1회 자기 평가 (1~10점, 각 영역)
- 레이더 차트로 시각화
- 월간 변화 트렌드 (이전 달 대비)
- 균형도 점수: 영역 간 표준편차 (낮을수록 균형)
- 불균형 영역 하이라이트

### 6.3 목표→Task 추적

- 목표별 투입 뽀모도로 시간 자동 집계
- 목표별 완료 Task 수 / 전체 연결 Task 수
- 목표별 주간 투입 시간 차트
- "이 목표에 연결된 Task가 없습니다" 경고 (전략적 갭 감지)

## 7. 화면 흐름

```
탭바 [자기관리]
  └─ 대시보드 (커스터마이징 가능)
      ├─ 일일 성과 점수
      ├─ 오늘 요약 카드
      ├─ 번아웃 경고 (조건 시)
      ├─ 에너지 체크인 (미완료 시 인라인)
      │
      ├─ [통계] 탭
      │   ├─ 시간대별 집중 히트맵
      │   ├─ 아이젠하워 시간 분배
      │   ├─ 계획 vs 실행 갭
      │   └─ 에너지 차트
      │
      ├─ [리뷰] 탭
      │   ├─ 하루 마감 리뷰
      │   ├─ 주간 리뷰
      │   ├─ 승리 기록
      │   └─ 주간 한 줄 요약 타임라인
      │
      ├─ [목표] 탭
      │   ├─ 분기 OKR
      │   ├─ 목표→Task 추적
      │   └─ Wheel of Life
      │
      └─ [습관] 탭
          ├─ 오늘의 습관 체크리스트
          ├─ 습관 관리 (생성/편집)
          ├─ 스트릭 현황
          ├─ 연간 히트맵
          └─ 습관-에너지 상관분석
```

## 8. 위젯

| 위젯 | 크기 | 내용 |
|------|------|------|
| 성과 점수 | Small | 오늘 점수 + 7일 미니 차트 |
| 에너지 | Small | 현재 에너지 레벨 + 체크인 버튼 |
| 습관 | Medium | 오늘 습관 체크리스트 (체크 가능) |
| 주간 요약 | Medium | 이번 주 집중/완료/에너지 요약 |
